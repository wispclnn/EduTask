# EduTask
a Flutter app designed to help students manage their tasks, deadlines, and schedules efficiently. It provides a clean and interactive interface, making it easy to track upcoming or overdue tasks, plan with a calendar, and customize the app's theme.

# Features

* Task Management: Add, edit, and remove tasks with a title, subject, description, and deadline.

* Upcoming & Overdue Tasks: View tasks closest to deadlines at a glance.

* Interactive Calendar: See all tasks due on a selected day.

* Live Search: Quickly find tasks using a live search with suggestions.

* Custom Themes: Change the app color to match your style.

* Persistent Storage: Tasks and theme preferences are saved automatically (JSON).
